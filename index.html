Here is a selection of some recent work.

justDance

This is a community app features real-time messaging for dancers. It allows dancers to create profile, add, edit and delete posts/review for dancing studios. Users are able to take advantage of the real-time messaging feature to leave messages, organize get-together and dancing events.

Stack: React, Redux, Node, Express, and MongoDB.

bonaPetite

A user can set a calories goal for one day, make meal plans, type in the name of the food, and the website will calculate the total calories of all the meals. This web app features an external API of US department of Agriculture, allowing users to check nutrition values of an ingredient provided by the API.

Stack: Javascript, Node, Express, and MongoDB.

wayfarer

This travel application is designed for users to create profile, posts, and share travel stories. It features a carousel (built with react bootstrap), a slide bar, and modals.

Stack: React, Node, Express, and MongoDB.

BarApp

This is a bartender point of sale app that allows a user to add drinks to a bar tab, save tabs by customer name, display balance, and search for cocktail recipes by an external API.

Stack: Python, Django, SQL and PostgreSQL.
